How It Works

Your agent asks. The stack delivers.

Agent wants to act
FrankenClaw serves the tool over MCP. Search the web, read an image, automate a browser, scrape a page — no extra config.
Agent wants to remember
Mnemo Cortex stores it with semantic indexing. Next session, next agent, next week — the memory is there.
You want to save money
Mnemo stops agents from re-doing yesterday's thinking. FrankenClaw uses your keys, not theirs. All local compute, zero cloud costs.
Install

Install the Full Stack

One command. Interactive menu. Works on Ubuntu and Mac. Five minutes or less.

One Command
curl -fsSL https://projectsparks.ai/install.sh | bash

The installer clones each repo, installs dependencies, walks you through API key setup, and wires everything together. If OpenClaw is detected, it offers to point your agent at the stack automatically.

If it's not easy, it's broken.
Safe to run again if something failed · Idempotent by design

Or install individually

Each product works on its own. Pick what you need.
