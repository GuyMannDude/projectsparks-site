Direct API

Bypass OpenRouter entirely

Paste your own vendor API key — Google, OpenAI, Anthropic, whoever. Rocky's Router talks directly to the vendor. No middleman markup. No routing fees.

PIN-secured so your keys stay safe. Keys persist across sessions. Switch vendors with one click.

PIN Security
Set a 4+ character PIN. Every key operation requires it. No one touches your vendor keys without the PIN.
Key Persistence
Save keys once, use forever. Switch between vendors without re-entering. Rocky remembers your keys so you don't have to.
Submarine Console

Built for operators, not dashboards

Dark UI. Sonar pings. Digital readouts. Wake event tracking. You're running a fleet, not reading a spreadsheet.

Real-time status for every tier. Model switching with immediate sync to your agent config. The console tells you what's happening and lets you change it in one click.
