Free by default. Smart when you talk.

Multiple Providers

OpenRouter, or go direct

OpenRouter is required for the free tier. But for your smart model, you can go direct — OpenAI, Anthropic, or Google. Your key, their API, no middleman.

Keys are stored locally with strict file permissions. Never in the repo. Never in a config file anyone can read.

Secure Storage
API keys stored in ~/.guysrouter/ with 600 permissions. Outside the project directory. Never committed to git.
Full Streaming
SSE passthrough for all providers. Works with OpenClaw, Claude Code, or any OpenAI-compatible client. No response buffering.
Simple UI

Pick your models. Drop your key. Done.

Clean dark dashboard at localhost:50070. Live stats showing free vs smart hits. Two dropdowns, one key field, one save button. No complexity. No three-tier confusion. No submarine console.

Runs as a systemd service. Starts on boot. Forgets nothing.
