🔧 What is ThreadLocker?

ThreadLocker enables any AI (ChatGPT, Claude, Grok, etc.) to:

  • Track up to 5 active conversation threads.
  • Show anchor summaries every 5 rounds.
  • Silently reconnect dropped topics.
  • Warn when the AI's memory buffer is full.
  • Use Session Saver for session backups.

It provides long-term memory without subscriptions or native memory features.

📋 How to Activate ThreadLocker

  1. Copy the Prompt: Find the ThreadLocker v2.1.1 prompt in .txt, HTML, or the Toolbox page.
  2. Paste Into the AI: Paste into your AI’s text box as the first message. Optionally say: "Start ThreadLocker now".
  3. Start Talking: The AI begins counting at [1/5], [2/5], ..., summarizing threads at [5/5].

📌 Pinning Threads (Lock Priority Topics)

To prioritize a thread, use Pinned Thread #1.

To pin a thread:

Pin 1 to [topic or project name]

Example: Pin 1 to Pass It On TOD project

To reset:

Reset pinned thread

Or: Pin 1 to [new topic]

🧱 Reading the Anchor Summary ([5/5] Rounds)

Every 5 rounds (question + response = 1 round), ThreadLocker shows:

[5/5] 2025-05-26 14:00 PDT — Anchor Summary:
THREAD 1: Pass It On – Alabama docs collected
THREAD 2: Server boot issues – drive not mounted
THREAD 3: Toolbox web layout cleanup
THREAD 4: ThreadLocker cross-AI testing notes
THREAD 5: Side ideas for Claude fallback logic

This confirms your conversation stays on track.

🚨 Overflow Detection (Buffer Warnings)

ThreadLocker monitors memory limits. If triggered:

🔹🔹🔹 BUFFER ALERT 🔹🔹🔹
❌ NEW CHATS MAY NOT RECORD ❌
SESSION SNAPSHOT or RESET recommended.

The AI may stop recording new messages. Back up or reset immediately.

💾 Session Saver: Save and Continue

Use Session Saver to back up sessions. Say: Save session now

Output example:

SESSION SAVER SNAPSHOT — 2025-05-26 14:10 PDT
THREADS:
- Pass It On – Deed forms for Arizona
- Server boot fix pending
GOALS:
- Finish top 5 state folders
- Finalize HTML export
NOTES:
- Claude test pending. Alex responded well.
- Save this block!
- To resume: Paste block, say “START FROM SNAPSHOT”

Paste into a new session to resume.

🧠 Best Practices

  • Use clear names for pinned threads.
  • Pin one thread at a time for Pinned Thread #1.
  • Let ThreadLocker auto-manage other threads.
  • Save Session Saver snapshots before resets.
  • Re-paste the prompt in free-tier AIs if memory resets.

🛠️ Need Help?

If the AI misbehaves:

  • Re-paste the prompt.
  • Ensure your AI supports long instructions.
  • Say: "Show ThreadLocker status" for a manual check.
