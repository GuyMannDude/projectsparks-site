Five endpoints. Total recall.

Mnemo Cortex sits alongside your agent as a memory sidecar. It captures every conversation, builds semantic indexes, and validates responses against what it remembers.

/context
Search memory before responding. Returns ranked chunks from the L1/L2/L3 cache hierarchy.
/preflight
Validate before sending. Cross-references drafts against memory. PASS, ENRICH, WARN, or BLOCK.
/ingest
The Live Wire. Captures every exchange as it happens. Under 5ms. Crash-safe.
/writeback
Archive sessions with summaries and key facts. Auto-indexed for future semantic search.
/sessions
List and inspect sessions across hot, warm, and cold tiers. Full transcripts on demand.

The Origin Story

The $100 Brain That Cured AI Amnesia

For two years, Guy Hutchins — a 73-year-old maker in Half Moon Bay, California — acted as the "Human Sync Port" for his AI agents. Every session, he'd manually copy conversation transcripts, paste context summaries back into fresh chats, and watch his agents forget everything the moment the window closed.

Then came OpenClaw, Rocky, and a $100 Claude subscription that changed everything. In one session, Guy, Rocky, and Opie (Claude) designed, built, and tested a lightweight memory coprocessor that actually worked. Four endpoints. Local Ollama models. No cloud dependency.

"I started with ChatGPT 3.5 and the first thing that mattered was memory. Two years of 'how do I make AI remember.' Now we're shipping the answer."

— Guy Hutchins, creator of Project Sparks
Latest Release

"Don't Fear the /new!"

Proven on live agents — Rocky with months of recall, plus Claude Code with cross-agent memory. The latest release features SQLite + FTS5, a context frontier with active compaction (80% compression, perfect recall), and DAG-based summary lineage so every summary traces back to its source.

SQLite + FTS5
All memory in a single database with full-text search. Fast, portable, zero dependencies.
Active Compaction
Rolling frontier. Older messages get summarized automatically — 80% compression, perfect recall.
Summary Lineage
Every summary tracks which messages built it via a DAG. Expand any summary back to verbatim source.
Sidecar Design
Observes session files from outside — never patches the agent. If Mnemo crashes, the agent keeps working.

Built Different

What makes Mnemo Cortex special

Live Wire
Session watcher captures every exchange automatically. If the plug gets pulled, nothing is lost.
Circuit Breaker
Provider fallback chains with automatic failover. If Ollama dies, it hot-swaps to your API backup.
Persona Modes
Strict mode for business. Creative mode for brainstorming. Or build your own.
Multi-Tenant
Each agent gets isolated memory. Filesystem-level isolation, not just namespaces.
Any Provider
Ollama (free/local), OpenAI, Anthropic, Google Gemini, OpenRouter, HuggingFace. Mix and match.
Any Framework
Claude Code, Claude Desktop, and OpenClaw integrations included. Or just use HTTP — five endpoints, any agent.
