What your guide can set up

Every deployment starts with OpenClaw. Everything else is your choice — the guides recommend these based on what works, but you pick what fits your needs.

OpenClaw
The foundation. Your AI agent framework, running on your hardware, under your control. This is the one non-negotiable.
Mnemo Cortex
Optional but recommended. Persistent memory so your agent remembers across sessions. 80% compression, perfect recall. Open source — or bring your own memory solution.
Rocky's Router
Optional. Three-tier model routing that saves money — free models for routine work, smart models only when needed. Direct API bypasses OpenRouter entirely. Saves real money on agent costs.
Channels
Your choice. Telegram, Discord, WhatsApp, web chat, Slack — connect whatever channels make sense for you. Start with one, add more anytime.
