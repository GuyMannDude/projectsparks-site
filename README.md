# projectsparks-site

Source code for **[projectsparks.ai](https://projectsparks.ai)** — the home of Project Sparks.

## What is Project Sparks?

An AI infrastructure and creative technology operation run by a non-developer, using a multi-agent assembly line (Claude, ChatGPT, OpenClaw) to build real tools and ship open-source software. Full story on the [GitHub profile](https://github.com/GuyMannDude).

## Stack

- **Hosting:** Firebase Hosting
- **Frontend:** Static HTML/CSS/JS
- **Analytics:** GA4
- **Pages:** Landing (constellation layout), Rocky's Router, Mnemo Cortex, FrankenClaw, Privacy, ToS
