The Moves

The v0.6.0 Dual-Engine Update

Every AI agent has amnesia. Mnemo Cortex fixes that with two engines.

The Semantic Vector Engine (architected by Opie) tracks live conversation state and finds fuzzy memories by meaning.

The Exact-Match Engine (architected by AL ✨) uses SQLite FTS5 for precise keyword recovery. It enables instant retrieval of things like "What is the Shopify API key?".

Four endpoints. Total recall.

Mnemo Cortex sits alongside your agent as a memory sidecar. It captures every conversation, builds semantic indexes, and validates responses against what it remembers — all through simple HTTP calls.

🔍
/context
Search memory before responding. "What do you remember about Easter?" Returns ranked chunks from the L1/L2/L3 cache hierarchy.
🛡️
/preflight
Validate before sending. Your agent drafts a response, Mnemo cross-references it against memory. PASS, ENRICH, WARN, or BLOCK.

The Live Wire. Captures every exchange as it happens. If the session crashes, everything is already on disk. Under 5ms.

The Origin Story

The $100 Brain That Cured AI Amnesia

For two years, Guy Hutchins — a 73-year-old maker in Half Moon Bay, California — acted as the "Human Sync Port" for his AI agents. Every session, he'd manually copy conversation transcripts between his PC and iPhone, paste context summaries back into fresh chats, and watch his agents forget everything the moment the window closed.

He tried everything. Forced transcript summaries (ThreadLocker). Reading QR codes off a monitor to transfer AI state between devices. Word documents ferried back and forth like carrier pigeons. Nothing stuck. AI memory was a problem everyone talked about and nobody solved.

"I started with ChatGPT 3.5 and the first thing that mattered was memory. Two years of 'how do I make AI remember.' Now we're shipping the answer."

— Guy Hutchins, creator of Project Sparks

Then came OpenClaw — the persistent terminal environment — and Rocky — the agent living inside it. and a $100 Claude subscription that changed everything. In one session, Guy, Rocky, and Opie (Claude) designed, built, and tested a lightweight memory coprocessor that actually worked. Four endpoints. Local Ollama models. No cloud dependency. Every conversation captured in real-time.

They named it Mnemo Cortex — the memory superhero who never forgets so your agent doesn't have to.

NEW RELEASE

"Don't Fear the /new!"

Your AI agent forgets everything between sessions. Mnemo Cortex fixes that. Now proven on two live agents — Rocky with six weeks of recall, Alice with one. v2.0 replaces JSONL files with SQLite + FTS5, introduces a context frontier with active compaction (80% compression, perfect recall), and adds DAG-based summary lineage so every summary traces back to its source.

SQLite + FTS5
All memory in a single database with full-text search. Fast, portable, zero dependencies. No more JSONL files.
Active Compaction
A rolling frontier of context items. Older messages get summarized automatically — 80% compression, perfect recall.
Summary Lineage
Every summary tracks which messages built it via a directed acyclic graph. Expand any summary back to verbatim source.
Sidecar Design
Observes session files from outside — never patches the agent. If v2.0 crashes, the agent keeps working. Clean separation.

Sparks Router v2.0

Don't burn tokens on heartbeats.

Every AI agent burns tokens on tasks that don't need expensive models. A heartbeat check doesn't need Opus. A quick lookup doesn't need Gemini Pro. But without routing, every request hits the same model at the same price.

Sparks Router is a task-based model routing proxy for OpenClaw agents. It reads your prompts, classifies the task complexity, and routes to the right model tier automatically. Three tiers. Automatic classification. Budget caps. You set the ceiling, the router handles the rest.

Three tiers. Automatic classification.

🧠
Smart Tier — $$$
Deep reasoning, architecture, complex analysis, multi-step planning. Routes to Gemini 3.1 Pro. Only fires when the request actually needs heavy thinking. The expensive model earns its keep.
⚙️
Utility Tier — $$
Code generation, summaries, structured tasks, tool calls, search. Routes to Gemini 2.5 Flash. Fast, cheap, and handles most of your agent's daily traffic.
🔓
Free Tier — $0
Heartbeats, cron jobs, health checks, lookups, formatting. Routes to Nemotron 120B (free via OpenRouter). Zero cost. Always. This is where the savings live.

What makes Sparks Router special

Keyword Classifier
Configurable trigger words per tier. "analyze" → Smart. "summarize" → Utility. "ping" → Free. Customize for your workflow.
Budget Caps
Set a daily spending limit per tier. Hit the cap? Requests auto-downgrade to free. Your agent keeps working. Your wallet stays closed.
Profile Presets
Save named routing configs. Standard Dive for daily work. Budget Dive for cost control. Full Throttle for critical work. Up to 10 slots.
Streaming
Full SSE streaming passthrough. Your agent gets responses token by token, just like talking to OpenRouter directly. Completely transparent.
Drop-In Proxy
Same API format as OpenRouter. Change one URL in your agent config. No code changes. Your agent doesn't even know the router exists.
Cost Tracking
Every response includes token counts and cost data. Track cumulative spending per tier, per day. Know exactly where your money goes.
Advanced

Consensus Engine (PMO)

Poor Man's Opie — when one model isn't enough. Fan a critical task out to multiple models and synthesize a consensus response. Four models argue, one judge arbitrates. Better answers than any single model for the decisions that matter most.
